DMARC 分析支援ツール マニュアル

このマニュアルは DMARC 分析支援ツール v1.0 の操作を章ごとに説明します。左のメニューから章を選択してください。

このツールの流れ

  1. 同意 — 初回起動時に利用規約・免責事項を確認し、同意する
  2. 取込 — フォルダまたは Microsoft 365 から DMARC レポート（XML / ZIP / GZ）を取得
  3. 解析 — 6 分類で自動判定（なりすまし疑い / 設定不備 / 転送由来 / 正常送信 / バウンス / 判定不能）
  4. 確認 — 一覧・フィルタ・トレンドで状況を把握
  5. エクスポート — CSV で外部ツール（Excel / BI）連携

マニュアルの範囲

v1.0 リリース時点では上記目次の 0 / 1 / 4 / 7 章のみを公開しています。「予定」と表示された章は、v1.0 リリース後の画面確定に合わせて順次公開します。当面はアプリ内ツールチップと、製品トップページの機能説明もあわせてご参照ください。

DMARC Analysis Assistant Manual

This manual walks through DMARC Analysis Assistant v1.0 chapter by chapter. Choose a chapter from the menu on the left.

Workflow

  1. Consent — Review the Terms of Use & Disclaimer on first launch and accept
  2. Import — Load DMARC reports (XML / ZIP / GZ) from a folder or from Microsoft 365
  3. Analyze — The six-category engine classifies each entry (Spoofing suspected / Config issue / Forwarded / Legitimate / Bounce / Unknown)
  4. Review — Inspect results with filters and trends
  5. Export — Export to CSV for Excel or BI tools

Scope of this manual

At the v1.0 release, only chapters 0 / 1 / 4 / 7 are published. Chapters marked "planned" will be added after the v1.0 UI is finalized. For now, please also consult the in-app tooltips and the feature overview on the product top page.
