マニュアル

DMARC 分析支援ツール

詳細なマニュアルは v1.0 Store 公開後に順次公開予定です。現時点では本ページ下部の概要と、アプリ内のツールチップ・ヘルプ文言をご参照ください。

1. インストール

  • Microsoft Store から「DMARC 分析支援ツール」を検索してインストールします。
  • Microsoft Edge WebView2 Runtime が必要です（Windows 11 には通常プリインストール）。

2. 初回起動時の同意

  • 初回起動時に「利用規約・免責事項」が表示されます。
  • 内容をご確認のうえ、「同意してはじめる」を選択するとアプリが利用可能になります。

3. DMARC レポートの取込

  • 取込元フォルダを指定して、XML / ZIP / GZ 形式の集約レポートを取り込みます。
  • 任意機能として Microsoft 365 連携を有効にすると、受信トレイから直接取り込めます（権限は Mail.Read のみ）。

4. 分類結果の確認

  • 6 分類ラベル（なりすまし疑い / 設定不備 / 転送由来 / 正常送信 / バウンス / 判定不能）で一覧表示されます。
  • ドメイン別、分類別、期間別のトレンドをグラフで確認できます。

5. CSV エクスポート

  • 分類結果を CSV で出力し、Excel / BI ツールで追加分析できます。

6. データの削除

  • アプリ内の「データ削除」メニューから削除できます。
  • または %LocalAppData%\DmarcAnalysisAssistant フォルダを手動削除、あるいはアプリをアンインストールしてください。

Manual

DMARC Analysis Assistant

Detailed manual pages will be published after v1.0 Store release. For now, please refer to the overview below and the in-app tooltips and help text.

1. Installation

  • Search for "DMARC Analysis Assistant" in Microsoft Store and install.
  • Microsoft Edge WebView2 Runtime is required (usually pre-installed on Windows 11).

2. First-launch consent

  • "Terms of Use & Disclaimer" is shown on first launch.
  • Review the terms and choose "Agree and continue" to start using the app.

3. Importing DMARC reports

  • Specify a source folder to import XML / ZIP / GZ aggregate reports.
  • Optionally enable Microsoft 365 integration to fetch directly from the inbox (scope: Mail.Read only).

4. Reviewing classification results

  • Six-category labels (Spoofing suspected / Config issue / Forwarded / Legitimate / Bounce / Unknown) are shown in the list view.
  • Trends by domain, category, and period are visualized in charts.

5. CSV export

  • Export classification results to CSV for further analysis in Excel or BI tools.

6. Data deletion

  • Use the in-app "Delete data" menu.
  • Or manually remove the %LocalAppData%\DmarcAnalysisAssistant folder, or uninstall the app.
