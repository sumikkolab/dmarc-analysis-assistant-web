よくあるご質問

Q. このツールは Microsoft 365 の設定を変更しますか？
A. 変更しません。Microsoft 365 への接続は読み取り専用で、使用権限は Mail.Read のみです。メールの送信・削除・編集も行いません。DNS 設定も変更しません。
Q. 解析対象のデータはクラウドに送信されますか？
A. 送信しません。レポート本文、IP アドレス、認証結果、ドメイン情報、分類結果は外部サーバーに送信されません。クラウド解析や外部 AI API にも問合せを行いません。
Q. Microsoft 365 連携は必須ですか？
A. 必須ではありません。連携を使わず、DMARC レポートファイル（XML / ZIP / GZ）を直接取り込んで利用することもできます。
Q. 6 分類の判定はどのように行われていますか？
A. ルールベースの分類エンジンが、レポート本文とメールヘッダーから判定します。AI 推論や機械学習は使用していません。同一入力に対して常に同一の分類結果を返します。
Q. 保存されたデータはどこにありますか？ 削除したい場合は？
A. 設定・解析結果は %LocalAppData%\DmarcAnalysisAssistant 配下に保存されます。アプリ内の「データ削除」メニュー、フォルダの手動削除、アプリのアンインストールのいずれでも削除できます。
Q. 認証トークンの扱いは？
A. Microsoft 365 の認証トークンはメモリ上にのみ保持し、ディスクには一切書き込みません。アプリを終了するとトークンは破棄されます。
Q. 「なりすまし疑い」と表示された場合、すぐに対応が必要ですか？
A. 判定は参考情報です。送信元 IP、認証結果、送信経路を確認したうえで、組織のセキュリティ担当者と相談して対応方針を決めてください。機械判定のため、すべての事例を正確に分類できるとは限りません（免責事項 第2条）。
Q. アプリが起動しない・WebView2 エラーが出る
A. Microsoft Edge WebView2 Runtime がインストールされているか確認してください（Windows 11 には通常プリインストールされています）。未インストールの場合は Microsoft の公式ページからインストールできます。
Q. バージョンアップの通知はありますか？
A. Microsoft Store 経由でインストールされた場合は Store の自動更新機能に従います。本サイトおよびアプリ内「このアプリについて」からも最新バージョンをご確認いただけます。

Frequently Asked Questions

Q. Does this tool modify Microsoft 365 settings?
A. No. Microsoft 365 connections are read-only with the Mail.Read scope. The tool does not send, delete, or edit mail, and does not change DNS settings.
Q. Is analysis data transmitted to the cloud?
A. No. Report contents, IP addresses, authentication results, domain information, and classification results are never sent to any external server. No cloud analysis or external AI API is used.
Q. Is Microsoft 365 integration required?
A. No. You can use the tool by directly importing DMARC report files (XML / ZIP / GZ) without enabling Microsoft 365 integration.
Q. How does the six-category classification work?
A. A rule-based classification engine analyzes report contents and mail headers. No AI inference or machine learning is used. The same input always produces the same classification result.
Q. Where is stored data kept, and how can I delete it?
A. Settings and analysis results are stored under %LocalAppData%\DmarcAnalysisAssistant. You can delete them via the in-app "Delete data" menu, by manually removing the folder, or by uninstalling the app.
Q. How are authentication tokens handled?
A. Microsoft 365 authentication tokens are held in memory only and are never written to disk. They are discarded when the application exits.
Q. If "Spoofing suspected" is shown, do I need to act immediately?
A. The classification is advisory. Review source IP, authentication results, and sending path, and consult with your security team. Since the result is machine-generated, not every case can be classified correctly (Disclaimer, Article 2).
Q. The app does not start or shows a WebView2 error.
A. Ensure Microsoft Edge WebView2 Runtime is installed (usually pre-installed on Windows 11). If not, you can install it from Microsoft's official page.
Q. How will I be notified of updates?
A. If installed via Microsoft Store, updates follow Store auto-update. You can also check the latest version on this website and under "About this app" in the application.
