プライバシーポリシー

最終更新: 2026年4月21日

本ソフトウェアはローカル端末上でのみ動作し、レポート内容・IP 情報・認証結果・ドメイン情報を外部に送信しません。テレメトリ・クラッシュレポート・使用履歴の自動送信も行いません。

プライバシー通知（アプリ内表示と同一内容）

  1. 本ソフトウェアはローカル端末上でのみ動作します。解析対象のレポートデータ、バウンスメール本文、設定情報を外部サービスへ送信することはありません。
  2. Microsoft 365 への接続は、利用者が設定画面で明示的に設定した場合に限り行われ、使用する権限スコープは Mail.Read のみです。メールの送信・削除・編集は行いません。
  3. Microsoft 365 の認証トークンはメモリ上に一時保持されるのみで、ディスクへの永続化は行いません。アプリを終了すると破棄されます。
  4. 分類結果、IP アドレス、認証結果、ドメイン情報などを外部サービスへ送信することはありません。クラウド解析や外部 AI API への問合せも行いません。
  5. 利用状況のテレメトリ、クラッシュレポート、使用履歴の自動送信・収集は行いません。
  6. 設定情報と解析結果はローカル端末の %LocalAppData%\DmarcAnalysisAssistant 配下に保存され、利用者の操作で削除できます。

補足説明

収集・保存するデータ

本ソフトウェアがローカル端末上に保存するデータは次のとおりです。いずれも %LocalAppData%\DmarcAnalysisAssistant 配下に保存され、アンインストールまたは利用者の操作で削除できます。

  • 取込んだ DMARC 集約レポート（XML / ZIP / GZ 展開後）
  • 取込んだバウンスメールのヘッダー・本文（ローカル DB 内）
  • 6 分類エンジンによる判定結果
  • アプリ設定（表示言語、取込フォルダ、同意状態のハッシュ等）

Microsoft 365 連携（任意機能）

利用者が設定画面で Microsoft 365 連携を明示的に有効化した場合に限り、Microsoft Graph API を用いて受信トレイから DMARC レポートおよびバウンスメールを取得します。

  • 使用権限スコープ: User.Read / Mail.Read（読み取り専用）
  • 送信・削除・編集は行いません。
  • 認証トークンはメモリ上にのみ保持され、ディスクには保存されません。アプリを終了すると破棄されます。
  • 連携を停止するには設定画面でサインアウトしてください。

外部サービスへの送信

分類結果、IP アドレス、認証結果、ドメイン情報、レポート本文を外部サービスへ送信することはありません。クラウド解析、外部 AI API、テレメトリ収集、広告配信のいずれも行いません。

データの削除

保存データは利用者の操作でいつでも削除できます。

  • アプリ内の「データ削除」メニュー（または設定画面の初期化）
  • エクスプローラーで %LocalAppData%\DmarcAnalysisAssistant フォルダを手動削除
  • Windows の「アプリのアンインストール」

第三者サービス

  • Microsoft 365 Graph API: 利用者が連携を有効化した場合のみ通信します。
  • Microsoft Edge WebView2 Runtime: GUI 表示に使用します。Microsoft のライセンスおよびプライバシー条項に従います。

プライバシーポリシーの変更

本ポリシーは予告なく変更されることがあります。重要な変更がある場合はアプリ内および本ページで通知します。

Privacy Policy

Last updated: April 21, 2026

This software runs locally on your device. Report contents, IP information, authentication results, and domain information are never transmitted externally. No telemetry, crash reports, or usage history are automatically collected or sent.

Privacy Notice (identical to the in-app text)

  1. This software runs locally on the user's device. Report data, bounce message contents, and configuration information are never sent to any external service.
  2. Microsoft 365 connections are made only when explicitly configured on the settings page, and the permission scope is limited to Mail.Read. The software does not send, delete, or edit mail.
  3. Microsoft 365 authentication tokens are held in memory only and are never persisted to disk. They are discarded when the application exits.
  4. Classification results, IP addresses, authentication results, and domain information are not sent to any external service. No queries are made to cloud analysis platforms or external AI APIs.
  5. Telemetry, crash reports, and usage history are not collected or transmitted automatically.
  6. Configuration and analysis results are stored under %LocalAppData%\DmarcAnalysisAssistant on the user's device and can be deleted by the user.

Supplementary Notes

Data collected and stored

The software stores the following data locally on your device. All data is stored under %LocalAppData%\DmarcAnalysisAssistant and can be deleted by uninstalling the app or through user actions.

  • Imported DMARC aggregate reports (XML, or extracted from ZIP/GZ)
  • Imported bounce message headers and body (in the local database)
  • Classification results produced by the six-category engine
  • Application settings (display language, import folder, consent state hash, etc.)

Microsoft 365 integration (optional)

Only when the user explicitly enables Microsoft 365 integration on the settings page, the software uses Microsoft Graph API to read DMARC reports and bounce messages from the inbox.

  • Permission scope: User.Read / Mail.Read (read-only)
  • The software does not send, delete, or edit mail.
  • Authentication tokens are held in memory only and are never persisted to disk. They are discarded on application exit.
  • To stop the integration, sign out from the settings page.

Transmission to external services

Classification results, IP addresses, authentication results, domain information, and report contents are never transmitted to external services. The software performs no cloud analysis, no external AI API calls, no telemetry collection, and no ad delivery.

Data deletion

Stored data can be deleted at any time by the user.

  • Using the in-app "Delete data" menu (or reset from the settings page)
  • Manually deleting the %LocalAppData%\DmarcAnalysisAssistant folder in Explorer
  • Uninstalling the app via Windows "Apps & features"

Third-party services

  • Microsoft 365 Graph API: used only when the user enables integration.
  • Microsoft Edge WebView2 Runtime: used for GUI rendering; subject to Microsoft's license and privacy terms.

Changes to this policy

This policy may be updated without prior notice. Significant changes will be announced in the application and on this page.
